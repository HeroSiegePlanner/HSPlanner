## New
- **Settings panel** - Manual save mode and number formatting (more to come)
- **Toggleable potions**
- **Merc tab** - Planner now supports mercs with magic find stats and auras affecting both player and merc
- **Ether Tree tab** - Like the incarnation tree but for ether, with stats summary and pathing support

- **Item conditions** - Items with conditional effects like `High Roller's Deathcrown` can now be toggled to include their bonus stats in calculations
- **Item picker** - Sort by DPS or any item stat, unified panels with net change and DPS hover preview
- **Auras** - Aura effectiveness affixes now buff aura output; skill auras are displayed
- Updated 600+ stats so most of the gear now should work
- Updated some items to the latest patch
